## 1. מטרה

לבנות משחק אינטראקטיבי ברמת web-browser שמלמד ילדים (גילאים מעורבים) לאתר מדינות ויבשות על מפת העולם — ללא ידע קודם נדרש.

---

- ילדים של דויד — גילאים מעורבים
- שפה: עברית (שמות מדינות, הוראות, UI הכל בעברית)
- גישה: דפדפן, ללא התקנה, ללא login

## 4. היקף תוכן ועיקרון הקושי הפרוגרסיבי

### עיקרון מרכזי: מהקל לאזוטרי
> השחקן **תמיד** מתחיל במדינות המוכרות ביותר, ועולה בהדרגה.
> המשחק לא שואל על מולדובה לפני שלמדת את ארה"ב.

### דירוג מדינות (Priority Score)
כל מדינה מקבלת דירוג פנימי (1–5):

| ציון | קטגוריה | דוגמאות |
|------|---------|---------|
| **5** | מדינות-ענק + ישראל שכנות | ארה"ב, רוסיה, סין, ברזיל, ישראל, מצרים, ירדן |
| **4** | מדינות מוכרות | צרפת, גרמניה, יפן, הודו, אוסטרליה, מקסיקו |
| **3** | מדינות מוכרות-בינוני | שבדיה, תאילנד, ניגריה, ארגנטינה |
| **2** | מדינות פחות מוכרות | קמבודיה, בחריין, בנין, גואטמלה |
| **1** | אזוטריות | קיריבאטי, נאורו, לסוטו, סנט קיטס |

### 4 רמות קושי — מבוסס על הדירוג
| רמה | מדינות שנשאלות | ציון דירוג | פתיחה |
|-----|----------------|-----------|--------|
| קלה | 7 יבשות בלבד | — | תמיד פתוחה |
| בינונית | ~50 מדינות | ציון 4–5 | פתוחה תמיד |
| קשה | ~130 מדינות | ציון 2–5 | פתוחה תמיד |
| מאסטר 🔒 | כל 195 | ציון 1–5 | **נפתחת אחרי סיום בינונית + קשה** |

> מאסטר = נעולה עד שמשחקים לפחות סיבוב אחד מוצלח בבינונית **וגם** בקשה.

### סדר שאלות בתוך סיבוב
- **תמיד** — שאלות ראשונות בסיבוב = ציון דירוג גבוה יותר
- כשמדינה נלמדת נכון 3 פעמים → עוברת לשאלה ב"בכיר" (מגיעה פחות)
- מדינות שטועים בהן → חוזרות יותר מהר (spaced repetition בסיסי)

## 5. שני מצבי משחק

### מצב A — "מצא את המדינה" (Click to Identify)
המשחק שואל: **"איפה ישראל?"**
המפה מוצגת ללא שמות. השחקן לוחץ על המדינה הנכונה.

**פידבק:**
- לחיצה נכונה → המדינה מתנוונת בירוק + מוצג כרטיס פידבק (שם + דגל)
- לחיצה שגויה → המדינה שנלחצה מתאדמת + המדינה הנכונה מוצגת בירוק + כרטיס פידבק

### מצב B — "מה שם המדינה?" (Name the Country)
מדינה מודגשת על המפה + 4 כפתורי תשובה.
השחקן בוחר את השם הנכון.

**פידבק:**
- תשובה נכונה → כפתור מוצהב/ירוק + כרטיס פידבק (שם + דגל)
- תשובה שגויה → כפתור שגוי מתאדם + הנכון מוצהב + כרטיס פידבק

### כרטיס פידבק (משותף לשני המצבים)
```
┌─────────────────────┐
│  🇮🇱  ✅ נכון!        │
│  ישראל              │
│  יבשת: אסיה         │
│  [המשך ←]           │
└─────────────────────┘
```
- טעות: אותה מבנה אך עם ❌ + "התשובה הנכונה היתה:"

## 6. זרימת משחק מלאה

```
מסך פתיחה (בחירת ילד / יצירת פרופיל)
    ↓
לוח ניקוד אישי (נקודות, מדינות שנלמדו, רמת שיא)
    ↓
בחירת מצב משחק: [מצא את המדינה] | [מה שם המדינה?]
    ↓
בחירת יבשת: [כל העולם] [אסיה] [אירופה] [אמריקה] ...
    ↓
בחירת רמה: [קלה - יבשות] [בינונית - 50 מדינות] [קשה - 195]
    ↓
סיבוב של 10 שאלות
    ↓        ↓
   נכון     טועה
  +10 נק׳    0 נק׳ + ריענון
    ↓
כרטיס פידבק (דגל + שם + יבשת)
    ↓
שאלה הבאה...
    ↓
מסך סיכום הסיבוב:
  - ניקוד הסיבוב
  - ניקוד כולל מצטבר
  - מדינות שנלמדו חדשות
  - האם הגיע לפרס? → 🎁 הצגת פרס
  - [שחק שוב] [יבשת אחרת] [בית]
```

## 8. מערכת פרופילים (מרובה ילדים)

**פרופילים ראשוניים:** ליבי (14.5) + בן (10.5) + כפתור "הוסף ילד"

**מסך פתיחה:**
```
┌─────────────────────────────┐
│  🌍 מי משחק?                │
│                             │
│  [🌺 ליבי]  [🚀 בן]         │
│  [+ הוסף ילד]               │
└─────────────────────────────┘
```

**מה שמור לכל פרופיל (localStorage):**
- שם + אווטאר emoji (בחירה מתוך 8 אפשרויות)
- סך נקודות מצטבר
- מדינות שנלמדו (רשימה + כמה פעמים נכון)
- שיא אישי לכל רמה
- פרסים שהושגו
- רמות שנפתחו

## 10. עיצוב ויזואלי

### סגנון: נקי ואינפוגרפי
- **פלטת צבעים:**
  - רקע: לבן / אפור בהיר
  - יבשות: 7 צבעי פסטל מובחנים (לא חזקים מדי)
  - הדגשה: כחול כהה / ירוק להצלחה / אדום לטעות
  - טיפוגרפיה: Heebo / Assistant (Google Fonts, עברית נקייה)
- **מפה:** קווי גבול עדינים, ים בגוון תכלת מאוד בהיר
- **UI:** כפתורים מעוגלים, צללית קלה, ללא זוויות חדות
- **דגלים:** emoji flags (🇮🇱) בכרטיס פידבק — פשוט וקל

### Responsive
| מסך | פריסה |
|-----|-------|
| דסקטופ (1280+) | מפה מרכזית + panel שאלה בצד |
| טאבלט (768–1280) | מפה מלאה + שאלה בתחתית |
| מובייל (<768) | מפה מקוצרת + שאלה בתחתית גדולה |

## 11. Technical Stack

| שכבה | כלי | הסבר |
|------|-----|-------|
| **מפה** | SVG (Natural Earth) + D3.js | כל מדינה = path SVG נפרד, ניתן ללחיצה + צביעה |
| **לוגיקה** | Vanilla JS | ללא build step, קל לפרוס |
| **עיצוב** | CSS variables | החלפת נושא קלה |
| **נתונים** | `countries-he.json` | שם עברי + ISO code + יבשת + emoji flag + **priority (1–5)** |
| **שמירה** | localStorage | פרופילים + ניקוד + התקדמות |
| **פריסה** | קובץ HTML יחיד + CSS + JS | עובד offline, קל לגיבוי |
